# Produkt — Instructions for Gemini

## Project
Product Context Engine — React 19 + TypeScript + Vite 7 + Tailwind v4

## Rules
- 3 colors only: #222121, #F9FEFF, #E2E7E9
- No TypeScript enums (use const objects)
- Engine has zero React dependency
- Port 3002

## Commands
- npm run dev → start
- npm run build → production build
- npm run lint → check types
